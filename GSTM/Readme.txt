App GSTM Version 1.0 05/01/2018

GENERAL USAGE NOTES
----------------------------------------------------------------------------------------
	- This is an application of an online shop
	- The application has a product search engine, to facilitate the 
	  user access to these.
	-It has a functional and operative shopping cart allowing the user to modify 
	  the quantity or elimination the products
	- The application allows to order the products according
	   to the choose of the user.
	- the application has two menus with which you can navigate through it

----------------------------------------------------------------------------------------
Minimum installation from Android API 15
----------------------------------------------------------------------------------------
The minimum version of Android that supports this application is API 15.
It works correctly for higher versions.
To make it work, it is recommended to use the Genymotion emulation
